# 🔥 SAPTHALA - Firebase System

## Quick Start

```bash
START.bat
```

That's it! The system will:
- ✅ Connect to Firebase/Firestore
- ✅ Start the server
- ✅ Open admin panel with charts

## What's Connected

- **Database**: Firebase/Firestore (Primary)
- **Charts**: Revenue Trend + Order Categories
- **Real-time**: All data syncs instantly
- **Status**: Check header for "Firebase: Connected"

## Verify

1. Dashboard shows statistics
2. Charts display below stats
3. Orders tab shows data
4. Green "Firebase: Connected" badge in header

## Done!

Your system is now running on Firebase with real-time charts.
