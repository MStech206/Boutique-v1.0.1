═══════════════════════════════════════════════════════════════════

1️⃣ Get Firebase Credentials
   └─ Download from: https://console.firebase.google.com
      Project: boutique-staff-app
      Save as: firebase-credentials.json

2️⃣ Install Packages
   └─ npm install firebase-admin mongoose cors

3️⃣ Create Migration File
   └─ File name: migrate.js
      (Code in MIGRATION_SIMPLE_GUIDE.md, Step 3)

4️⃣ Run Migration
   └─ node migrate.js

   └─ Check Firebase Console for collections

❓ COMMON QUESTIONS
═══════════════════════════════════════════════════════════════════

Q: Do I need to know programming?
A: No, just follow the steps. Copy-paste the code.

Q: Will it delete my MongoDB data?
A: No, it only copies to Firebase. MongoDB stays unchanged.

Q: How long does it take?
A: About 12 minutes total (5 min setup + 2 min migration + 5 min verify)

Q: What if something goes wrong?
A: Check Troubleshooting in MIGRATION_SIMPLE_GUIDE.md

Q: Do I need MongoDB running?
A: No, the script works with or without it.

Q: What's firebase-credentials.json?
A: Your secret key to access Firebase. Download from Firebase Console.
