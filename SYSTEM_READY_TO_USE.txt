🔍 TROUBLESHOOTING (If Needed):

Problem: "Cannot connect to server"
Solution: 
  1. Check MongoDB: Get-Process mongod
  2. Check port 3000: netstat -ano | Select-String "3000"
  3. Restart server: node server.js

Problem: "Login says invalid credentials"
Solution:
  1. Verify database: Check User collection in MongoDB
  2. Use exact password: sapthala@2029 (case-sensitive)
  3. Check server logs for bcrypt errors

Problem: "Staff portal shows no tasks"
Solution:
  1. Verify staff member exists in database
  2. Check orders are assigned to that staff
  3. Reload page: Ctrl+F5 (hard refresh)

Problem: "Images not loading"
Solution:
  1. Check /uploads directory exists
  2. Check /sapthala admin imgs directory exists
  3. Verify file paths in database
  4. Try different browser/clear cache
