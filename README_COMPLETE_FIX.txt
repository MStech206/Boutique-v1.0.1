🆘 IF SOMETHING GOES WRONG
═══════════════════════════════════════════════════════════════════

1. First check: Read FIREBASE_MIGRATION_GUIDE.md (Troubleshooting)

2. Run diagnostics:
   node diagnose-firebase.js
   
   This performs 7-point health check and tells you exactly
   what's wrong

3. Common issues & quick fixes:

   Admin panel shows 0 data:
   → Hard refresh: Ctrl+Shift+R
   → Wait 10 seconds
   → Clear cache: Ctrl+Shift+Delete

   "Forbidden" error:
   → Logout and login again
   → Check email matches credentials

   Server won't start:
   → Port 3000 might be in use
   → Check firebase-credentials.json exists
   → Re-run COMPLETE_FIX.bat

   Firebase connection error:
   → Check internet connection
   → Verify credentials file is valid JSON
   → Try again in 30 seconds
