**Solution**: 
1. Check Node.js installed
2. Check MongoDB running
3. Check port 3000 available
4. Run `node server.js` first

### Issue: Category not working
**Solution**: Clear browser cache (Ctrl+Shift+Delete)

### Issue: Images not showing
**Solution**: Verify `/images/` folder exists
